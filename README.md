# Hang in There  

### Abstract:
[//]: <> (Briefly describe what you built and its features. What problem is the app solving? How does this application solve that problem?)
I built a front end HTML that generates random posters, creates posters, saves posters, and able to look at unmotivational posters as well as delete them. The problem that was first announced was that there was a web page that had buttons that didn't work showed no poster, title, or quote.
The features that i built to make this application work was JavaScript, CSS, and HTML. The sections of HTML can be accessed by clicking the corresponding button giving you access to the hidden sections. In addition being able to create posters and using those images, titles, and quotes in random generated posters. Making additional features to make all applications work within the HTML page.
